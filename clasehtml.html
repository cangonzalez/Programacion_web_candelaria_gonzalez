Comandos con Efectos Visuales en HTML

A continuación, se muestran algunos comandos de HTML que tienen efectos visuales:

  • <b>: Texto en negrita
    • Este texto está en negrita
  • <i>: Texto en cursiva
    • Este texto está en cursiva
  • <u>: Texto subrayado
    • Este texto está subrayado
  • <mark>: Texto resaltado
    • Este texto está resaltado
  • <small>: Texto más pequeño
    • Este texto es más pequeño
  • <del>: Texto tachado
    • Este texto está tachado
  • <ins>: Texto subrayado (insertado)
    • Este texto está subrayado
  • <strong>: Texto importante (negrita)
    • Este texto es importante
  • <em>: Énfasis en el texto (cursiva)
    • Este texto tiene énfasis
  • <h1> a <h6>: Encabezados

    • Encabezado 1

    Encabezado 2

    Encabezado 3

    Encabezado 4

    Encabezado 5
    Encabezado 6
  • <p>: Párrafo

    • Este es un párrafo de ejemplo.

  • <blockquote>: Cita en bloque
    • Esta es una cita en bloque.
  • <hr>: Línea horizontal
  • <br>: Salto de línea
    • Línea 1
    Línea 2
